Project Goals

I created GameDevUtils.com for my game programming students to have access to high-quality, free tools. The goals of the project are to create tools that rival the for-pay apps in functionality and design, to provide access to the source code for those tools, and to get my students excited about game programming by making many of the more tedious production tasks easier.
